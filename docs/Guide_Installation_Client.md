# Guide d'Installation - Client The Tailor 🚀

Félicitations pour l'acquisition de votre nouveau système de gestion ! Ce guide vous aidera à installer et activer votre application en quelques minutes.

## 1. Installation de l'Application
1.  **Téléchargement** : Récupérez le fichier `the-tailor_1.0.0_x64-setup.exe`.
2.  **Lancement** : Double-cliquez sur le fichier pour lancer l'assistant d'installation.
3.  **Progression** : Suivez les instructions à l'écran et cliquez sur "Suivant".
4.  **Terminer** : Une fois terminé, une icône apparaîtra sur votre bureau et dans le menu Démarrer.

## 2. Première Ouverture & Connexion
-   Lancez l'application depuis votre bureau.
-   Vous accéderez à l'écran d'accueil "Bienvenue".
-   Cliquez sur **Administration** ou **Tableau de bord**.
-   Connectez-vous avec vos identifiants fournis séparément.

## 3. Activation de la Licence (Sécurité)
Lors du premier lancement ou après un renouvellement, le système de sécurité vous demandera une **Clé d'activation**.
1.  Sur l'écran de blocage, vous verrez un code nommé **ID Terminal (HWID)**.
2.  **Copiez ce code** (un bouton "Copier" est disponible) et envoyez-le à votre administrateur.
3.  Votre administrateur vous fournira une **Clé d'Activation**.
4.  Collez la clé dans le champ prévu et cliquez sur **Activer l'Accès**.
